# FullSubNet

This Git repository for our paper "FullSubNet: A Full-Band and Sub-Band Fusion Model for Real-Time Single-Channel Speech Enhancement" submitted to
ICASSP 2021. [[Paper]](https://arxiv.org/abs/2010.15508)
